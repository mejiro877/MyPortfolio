## ひとこと掲示板

- 概要<br>
情報共有を目的とした掲示板サイト<br>
名前やひとことに何も入力されていないとエラーを出力して再入力を促す仕様<br>
またひとこと仕様のため文字数制限有り<br>

## 自動販売機管理ツール

- 概要<br>
自動販売機の管理を想定したツール<br>
商品名と値段を入力し、画像ファイルをアップロードして追加すれば、<br>
図のように商品を追加することができる。商品データはMySQLで管理している。<br>
